# API REST para Gestión de Citas Médicas

API REST desarrollada en ASP.NET Core 8.0 para la gestión de citas médicas, utilizando MySQL como base de datos y JWT para autenticación.

## Requisitos Previos

- .NET SDK 8.0
- MySQL Server 8.0 o superior
- Visual Studio Code o cualquier IDE compatible
- Navicat (opcional, para gestión de BD)
- Thunder Client o Postman (para pruebas)
